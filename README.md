## Deployment Options

### Option 1: GitHub Pages
1. Upload all files to a GitHub repository.
2. Enable GitHub Pages in repository settings.
3. Share the public link.

### Option 2: Netlify
Drag and drop this folder into Netlify for instant hosting.

This is a fully client-side static application.
